Create Table (With Constraints Like Identity, Not Null, Default, PK, FK, Range)

Project Table:

	* ProjectID is an Identity column starting from 1 and incrementing by 1, serving as the Primary Key.
	* ProjectName must be unique.
	* Status default value of 'Not Started'.
	* A CHECK constraint ensures that EndDate (if provided) is after StartDate.

Task Table:

* TaskID is an Identity column starting from 1 and incrementing by 1, serving as the Primary Key.
* Priority Value must be one of 'Low', 'Medium', or 'High'.
* Status default value of 'Pending'.
* ProjectID references the ProjectID column in the Project table as a Foreign Key.
* A CHECK constraint ensures that DueDate (if provided) is after StartDate.
